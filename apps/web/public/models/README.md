# Sketchfab Models

Download the free Sketchfab models below as **glTF/GLB**, rename them to the listed filenames,
and place them in this folder.

## Required files

1. Abandoned Factory Modular Kit  
   https://sketchfab.com/3d-models/abandoned-factory-modular-kit-ec040b2034e949ffbc106f0784deb3fc  
   Filename: `abandoned_factory_modular_kit.glb`

2. Industrial robotic arm  
   https://sketchfab.com/3d-models/industrial-robotic-arm-abdddefccbc64dccbe6d8904664acbaa  
   Filename: `industrial_robotic_arm.glb`

3. Hologram Console  
   https://sketchfab.com/3d-models/hologram-console-bfbbb481e98e4be38774b1d0204c192c  
   Filename: `hologram_console.glb`

4. HEDGEHOG ISLAND LABOLATORY Fan Art  
   https://sketchfab.com/3d-models/hedgehog-island-labolatory-fan-art-a036aa74dd234c07a3a2af19cff18dc6  
   Filename: `hedgehog_island_labolatory_fan_art.glb`

5. EARTH  
   https://sketchfab.com/3d-models/earth-5ce4b1465c83432d9bb7e3c30232c02b  
   Filename: `earth.glb`
